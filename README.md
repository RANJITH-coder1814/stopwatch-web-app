# stopwatch-web-app
A lightweight stopwatch application that can measure elapsed time. Users can start, pause, and reset the timer easily. This project is great for beginners learning JavaScript timing functions.
# ⏱️ Stopwatch Web App

A simple stopwatch built using HTML, CSS, and JavaScript.

## Features
- Start timer
- Pause timer
- Reset timer
- Clean UI

## Technologies
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
